## What
Build a content enrichment pipeline that generates rich, legitimate listing content for every care home before launch. Combines CIW inspection report PDFs, the care home's own website, and Google Places API into bilingual descriptions generated by Claude.

## Why
CIW registration data alone gives us name, address, phone, and bed count. That makes for empty, useless listing pages. This pipeline turns each page into genuinely useful content for families — before a single care home claims their listing.

## Sources (all legitimate)

1. CIW inspection report PDFs — Open Government Licence, freely available
   Each report contains 4-8 pages of inspector narrative about the home

2. Care home's own website — their own public marketing content
   URL from care_homes.website column after CIW import
   Fetch homepage, /about, /services pages

3. Google Places API — free tier, explicitly permitted via API terms
   Returns: business description, user photos, rating, reviews
   Free tier: $200 credit/month covers the full dataset easily

## Acceptance criteria
- [ ] Migration adds: google_place_id, google_rating, google_review_count, google_photos[], enrichment_status, enrichment_date
- [ ] Script reads all care homes from Supabase where enrichment_status = 'pending'
- [ ] CIW report PDF: downloaded, text extracted via pdf-parse, narrative summarised by Claude
- [ ] Care home website: homepage fetched, key copy extracted, Claude pulls key info
- [ ] Google Places: searched by name + postcode, place_id stored, photos fetched (up to 3)
- [ ] Google photos saved to Supabase Storage at care-homes/[id]/google/[n].jpg
- [ ] Claude generates 150-word Welsh description combining all sources (natively Welsh, not translated)
- [ ] Claude generates 150-word English description

- [ ] Script is resumable — skips enrichment_status = 'complete'
- [ ] Rate limits respected: 1 req/2s CIW, 1 req/1s websites, Google quota managed
- [ ] Error log: /data/ciw/enrichment-errors/YYYY-MM-DD.json
- [ ] Summary printed: X enriched, X failed, X skipped
